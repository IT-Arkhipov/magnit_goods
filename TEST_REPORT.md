### ✅ Task 1.3: Добавлена retry логика

**Реализация:**
- 3 попытки получения товаров с exponential backoff (2, 4, 8 секунд)
- Логирование каждой попытки
- Пропуск категории после исчерпания попыток (не блокирует сканирование)
- Обработка различных типов ошибок API

**Код:**
```python
max_retries = 3
retry_count = 0
while retry_count < max_retries:
    try:
        result = self.api.get_products(...)
        break  # Успешно
    except Exception as e:
        retry_count += 1
        if retry_count < max_retries:
            wait_time = 2 ** retry_count
            time.sleep(wait_time)
```
