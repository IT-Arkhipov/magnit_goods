## 2. Отображение старой цены и улучшение фильтра на странице Товары

**Коммит:** `b347102`  
**Сообщение:** `feat: добавить отображение старой цены в tooltip и улучшить фильтр 'Экономия'`

### Что было изменено:

#### 1. Отображение старой цены в tooltip
- При наведении на цену товара показывается старая цена в tooltip
- Работает для всех магазинов (основного и сравниваемых)
- Tooltip содержит: "Старая цена: XXX ₽"

#### 2. Процент скидки в зелёном цвете
- Скидка отображается в скобках: `(-25%)`
- Цвет скидки: зелёный `#28a745`
- Показывается, если есть старая цена
- Пример: `189.99 ₽ (-14%)`

#### 3. Выделение цен красным
- Минимальная цена между магазинами выделена красным `#e31e25`
- Цены со скидкой выделены красным
- Это помогает быстро найти выгодные предложения

#### 4. Убрали колонки "Старая цена" и "Скидка"
- Таблица стала компактнее
- Информация о старой цене доступна в tooltip при наведении
- Процент скидки отображается прямо рядом с ценой

#### 5. Улучшен фильтр "Экономия (%)"
Фильтр теперь работает по двум условиям (OR логика):

**Условие 1:** Скидка от старой цены >= выбранного значения
- Проверяется в каждом магазине
- Если хотя бы в одном магазине скидка >= выбранного значения, товар показывается

**Условие 2:** Экономия между магазинами >= выбранного значения
- Вычисляется как процент разницы между минимальной и средней ценой
- Если экономия >= выбранного значения, товар показывается

Пример:
- Выбран фильтр "Экономия 20%"
- Товар показывается, если:
  - Скидка в любом магазине >= 20% (от старой цены), ИЛИ
  - Экономия между магазинами >= 20%

#### 6. Загрузка старой цены для всех магазинов
- При загрузке цен других магазинов теперь загружается и `old_price`
- Сохраняется в `otherOldPrices` для каждого магазина
- Используется для расчёта скидки и отображения в tooltip
